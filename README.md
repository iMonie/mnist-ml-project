# MNIST ML Project

This project builds and compares machine learning models on the MNIST dataset.

## Models Used
- CNN (TensorFlow) @20 epoch
- Logistic Regression
- Random Forest

## How to Run

Open in Google Colab:
https://colab.research.google.com

Upload `mnist_project.ipynb` or any of the codes and run all cells.

---

## Features
- Image classification (handwritten digits)
- Data augmentation
- Model comparison
- Confusion matrices
- Accuracy visualization

---

## Tech Stack
- Python
- TensorFlow
- scikit-learn
- NumPy
- Matplotlib
